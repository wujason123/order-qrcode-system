订单二维码查询系统 - 快速开始指南

🎯 系统功能：
- 读取Excel订单数据
- 生成订单二维码
- 提供Web查询界面
- 手机扫码查询订单
- 订单号重复检测
- 分页显示订单列表
- 二维码打印功能（Excel导出 + 网页打印）

📋 使用步骤：

方法一：一键启动（推荐）
1. 双击运行"启动系统.bat"文件
2. 系统会自动安装依赖包
3. 选择"1"一键启动完整系统
4. 访问 http://localhost:5000

方法二：分步安装（遇到网络问题时推荐）
1. 双击运行"安装依赖.bat"文件
2. 等待依赖包安装完成
3. 双击运行"启动系统.bat"文件
4. 选择"1"一键启动完整系统

方法三：手动运行
1. 安装依赖：pip install -r requirements.txt
2. 运行：python run.py
3. 选择相应功能

📁 文件说明：
- orders.xlsx: Excel订单数据（自动创建示例）
- qrcodes/: 二维码图片存储目录
- orders.db: SQLite数据库
- templates/index.html: 查询网页

🔧 自定义数据：
替换orders.xlsx文件，包含以下列：
- 订单号（必须唯一，不能重复）
- 客户姓名  
- 订单日期
- 金额（数字格式）
- 产品详情

⚠️ 重要提醒：
- 订单号在Excel文件内不能重复
- 订单号不能与数据库中已有订单重复
- 如有重复，系统会提示并提供删除选项

📱 使用方式：
1. 扫描qrcodes文件夹中的二维码
2. 在网页上手动输入订单号查询
3. 浏览所有订单列表

🖨️ 打印二维码：
方式一（推荐）：
1. 点击"Excel打印"按钮下载Excel文件
2. 用Excel打开文件并设置打印参数
3. 打印高质量的二维码标签

方式二（快速）：
1. 勾选要打印的订单复选框
2. 点击"网页打印"按钮
3. 在预览窗口中直接打印

❓ 遇到问题？
- 依赖安装失败：查看"故障排除指南.txt"
- 其他问题：查看 README.md 了解详细说明

📱 网络配置重要提醒：
手机需要与电脑在同一WiFi网络下才能扫码！

🔧 网络配置步骤：
1. 确保电脑和手机连接同一WiFi
2. 启动系统会显示手机访问地址
3. 手机浏览器先测试能否访问该地址
4. 网络正常后再扫描二维码

💡 常见问题快速解决：
- 依赖安装：使用"安装依赖.bat"脚本
- HTTP 403错误：更换镜像源安装
- 网络问题：查看"网络配置指南.txt"
- 防火墙：允许Python程序通过防火墙
